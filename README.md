# Calculator Application

This is a simple calculator application built with HTML, CSS, and JavaScript.

## Features

- Basic arithmetic operations: addition, subtraction, multiplication, and division.
- Clear button to reset the calculator.
- Delete button to remove the last entered digit or operator.

## How to Use

1. Clone the repository: `git clone https://github.com/crsarath00-ctrl/gemini.git`
2. Open `index.html` in your web browser.
